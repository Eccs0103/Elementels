Settings

Theme

External design of the page: color scheme, background and foreground, font.

Counter FPS

Show or hide FPS counter with load indicator.

Absolute frames per second

The absolute number of frames per second that is calculated for the speed of the interaction of elements, as well as to calculate the number of frames per second.

Elements counter

Show or hide the elements counter, which displays the icons of the elements, their names and quantities.

Hide nullables

Show or hide elements with zero count.

Board size

Board generated size. The size can be from 20 to 200 inclusive. Resizing the board will generate it anew with the given coefficients.
